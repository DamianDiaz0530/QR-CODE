📌 Descripción
En esta actividad desarrollarás un componente de Código QR utilizando Astro y Tailwind CSS, con el objetivo de poner en práctica tus conocimientos en la creación de componentes, uso de estilos y diseño responsivo.

El reto consiste en recrear el componente del Código QR lo más parecido posible al diseño proporcionado. Este ejercicio te permitirá fortalecer tus habilidades en el manejo de estructuras semánticas con HTML, así como la organización y estilo visual mediante Tailwind CSS.

Podrás apoyarte en el diseño de referencia (versión móvil y de escritorio), tomando tus propias decisiones sobre tamaños, márgenes, tipografía y espaciado.

🎯 Objetivos de aprendizaje
Practicar la creación de componentes utilizando Astro.

Aplicar conceptos de diseño responsivo mediante Tailwind CSS.

Reforzar el uso de HTML semántico y estructura de contenido.

Aprender a replicar un diseño visual con atención al detalle.

Desarrollar la capacidad de organizar y documentar su proyecto en un repositorio de GitHub.

✅ Requerimientos del reto
Tu proyecto deberá incluir los siguientes elementos:

Componente QR Code diseñado según la maqueta proporcionada.

Estructura semántica en HTML para el contenido del componente.

Estilos implementados con Tailwind CSS, cuidando la tipografía, márgenes y espaciado.

Diseño responsivo, visible correctamente en versión móvil y escritorio.

Imagen o ícono del código QR centrado dentro del componente.

Encabezado y texto descriptivo debajo de la imagen, tal como se muestra en el diseño.

Uso del archivo style-guide.md como referencia para paleta de colores y fuentes.

El componente debe mostrarse correctamente dentro de una página principal del proyecto.

🧠 Recomendaciones
Antes de comenzar:

Analiza el diseño y planea la estructura de tu componente.

Escribe primero la estructura base con HTML antes de aplicar los estilos.

Aplica estilos de arriba hacia abajo, asegurando que cada sección quede completa antes de continuar.

Ajusta las medidas de acuerdo a tu propio criterio (usa padding, margin y font-size adecuados).

Recuerda que puedes apoyarte en la documentación oficial de Astro y Tailwind CSS
